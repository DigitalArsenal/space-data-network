The Problem

Space is booming.
Data sharing isn't.

10,000+ Active satellites
40,000+ Tracked objects
30% Annual growth rate
  • ✗ Proprietary silos — Incompatible formats, bilateral agreements, email-based data exchange between operators
  • ✗ Single points of failure — Centralized clearinghouses can go down, get defunded, or become geopolitically contested
  • ✗ Legacy formats — TLEs were designed for punch cards in the 1960s. CDMs are exchanged as flat text files
  • ✗ $500K/yr software — Orbital mechanics tools like STK and FreeFlyer are inaccessible to most organizations
  • ✗ No data marketplace — No trustless, automated way to buy and sell space observations
The Solution

The TCP/IP of space.

A decentralized peer-to-peer protocol for exchanging standardized space data — with an integrated marketplace and browser-native tools that make space operations accessible to everyone.

🌐

Decentralized P2P

Like BitTorrent for space data. No central server, no single point of failure. Built on IPFS/libp2p.

📋

127 Standards

FlatBuffers-based schemas for every space data type. 13 programming languages. Zero-copy performance.

🚀

Runs in Your Browser

Full astrodynamics simulation via WebAssembly. No installs, no license servers, no vendor lock-in.

We built TCP/IP for space, and we're selling the Cisco routers.
